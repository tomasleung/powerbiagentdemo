# Outputs
## Semantic Design Agent

---

# Purpose

This folder contains the official outputs generated by the Semantic Design Agent.

These outputs transform:

```text
Technical Design

↓

Semantic Design
```

and provide the complete semantic design package required for downstream semantic model implementation.

---

# Output Sequence

Outputs must always be generated in the following order:

```text
DATA_MODEL_MATRIX

↓

SEMANTIC_MODEL_SPEC

↓

MEASURE_CONTRACT
```

Each output builds upon the previous output.

Never skip an output.
