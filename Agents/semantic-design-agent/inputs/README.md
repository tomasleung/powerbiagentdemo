# Inputs
## Semantic Design Agent

---

# Purpose

This folder contains all inputs used by the Semantic Design Agent.

Inputs provide the business intent, technical design requirements, and traceability needed to generate semantic design artifacts.

The Semantic Design Agent transforms:

```text
Technical Design

↓

Semantic Design
```

using the approved inputs contained within this folder.

---

# Input Responsibilities

The input artifacts define:

```text
Business Decisions

Business Questions

Signals

Measures

Technical Requirements

Source Systems
```

Inputs do not define:

```text
Semantic Architecture

Fact Tables

Dimension Design

Relationship Design

Measure Governance
```

Those are generated by the Semantic Design Agent.
