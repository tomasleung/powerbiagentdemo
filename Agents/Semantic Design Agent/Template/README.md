# Semantic Design Agent Template Files

This folder contains the template artifacts that define the expected structure for the Semantic Design Agent outputs.

## Purpose

These template files are used by the agent to understand the required sections and format for each output.

## Files

- `0 DATA_MODEL_STANDARDS_v1.0`
- `1 DATA_MODEL_MATRIX_v1.0`
- `2 SEMANTIC_MODEL_SPEC_v1.0`
- `3 MEASURE_CONTRACT_v1.0.md`

## How to use

1. Review the templates before generating outputs.
2. Use the templates to guide section names, artifact structure, and content expectations.
3. Apply the same structure when producing new versions of:
   - `DATA_MODEL_MATRIX_v1.0`
   - `SEMANTIC_MODEL_SPEC_v1.0`
   - `MEASURE_CONTRACT_v1.0`

## Relationship to the Runnable Spec

The runnable spec references these template files so the agent knows the expected output shape.
Use them as the first specification example before generating new artifacts from input documents.
