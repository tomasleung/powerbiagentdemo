# Templates Directory
## TRD Agent v1.0

---

# Purpose

This folder contains all approved templates used by the TRD Agent.

Templates define:

```text
Document Structure

Sections

Placeholders

Expected Output Format

Required Deliverables
```

Templates do not define:

```text
Business Requirements

Design Standards

Documentation Rules

Implementation Guidance
```

These are provided by:

```text
TRD_STANDARDS_v1.0

TRD_GUIDELINES_v1.0
```

# Relationship To Standards

Templates define:

```text
Structure
```

Standards define:

```text
Rules
```

Example

```text
TRD_TEMPLATE

defines

Where source systems are documented.
```

```text
TRD_STANDARDS

defines

What source information must be documented.
```

---

# Relationship To Guidelines

Templates define:

```text
Output Format
```

Guidelines define:

```text
Writing Style

Level Of Detail

Documentation Boundaries
```

Example

```text
TRD_TEMPLATE

defines

The Measure Inventory section.
```

```text
TRD_GUIDELINES

defines

How measures should be documented.
```
