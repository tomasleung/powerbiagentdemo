# Examples Folder

## Purpose

Contains approved reference implementations of the Decision Story Agent.

Examples demonstrate how the agent should behave when processing real business requirements.

Examples serve as:

```text
Reference Implementations

Training Examples

Validation Samples

Architecture Examples
```

# Governance

Examples should only contain:

```text
Approved Inputs

Approved Outputs

Approved Standards
```

Examples must never contain:

```text
Draft Artifacts

Experimental Structures

Incomplete Deliverables
```
